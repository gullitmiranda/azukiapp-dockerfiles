# Dockerfiles

The repository contains Dockerfiles for various public Docker images, namely:

* `azukiapp/node:0.10`:
  - Node 0.10.33
  - NPM 2.1.11

* `azukiapp/php-apache:5.6`:
  - Apache2
  - PHP Version 5.5.9
  - Composer

* `azukiapp/php-apache:5.5`:
  - Apache2
  - PHP Version 5.6.3
  - Composer

* `azukiapp/ruby:2.1`
  - Ruby 2.1.2
  - RubyGems 2.3.0
  - Bundler
  - ImageMagick

* All images contains:
  - Git
  - MySQL Client
  - PostgreSQL Client
  - MongoDB
